# Biente.Shop

### Project Setup
```bash
git clone https://github.com/meteoguzhan/biente-shop.git
```
```bash
cd biente-shop && composer install
```
```bash
php artisan migrate
```
```bash
php artisan serve
```
### Biente Api Get
```bash
http://localhost:8000/bienteApiGet
```
### Biente Api Post
```bash
http://localhost:8000/bienteApiPost
```
### Products Of Users
